# Carta de cumpleaños

Una pequeña carta virtual de cumpleaños creada con HTML, CSS y JavaScript para celebrar a alguien especial.

## Descripción

Este proyecto combina una página de felicitación con una animación de pastel de cumpleaños y una sección de regalo, para ofrecer una experiencia más emotiva y personalizada.

## Archivos principales

- `index.html`: página principal de la carta.
- `carta.css`: estilos de la carta.
- `carta.js`: interacciones de la carta.
- `birthday-cake/`: implementación del pastel animado de cumpleaños.
- `Regalo/`: sección con el regalo.

## Cómo verlo

1. Abre el proyecto en tu navegador.
2. Entra a `index.html` para ver la carta completa.
3. Si quieres probar la animación del pastel, revisa la carpeta `birthday-cake/`.
